ğŸ†˜ TROUBLESHOOTING
------------------
Q: ModuleNotFoundError: No module named 'django'
A: Make sure virtual environment is activated and run: pip install -r requirements.txt

Q: No module named 'PIL'
A: Install Pillow: pip install Pillow

Q: Page shows "No such table" error
A: Run migrations: python manage.py makemigrations && python manage.py migrate

Q: Static files not loading
A: Make sure DEBUG = True in settings.py for development

Q: Can't login to admin
A: Create superuser: python manage.py createsuperuser
